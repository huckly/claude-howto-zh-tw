# Claude How To - 設計系統

## 視覺識別

### 圖示設計概念：指南針搭配程式碼括號

Claude How To 圖示使用**指南針搭配 `>` 程式碼括號**來代表程式碼導覽的引導：

```
     N (綠色)
     ▲
     │
W ───>─── E     指南針 = 引導/方向
     │          > 括號 = 程式碼/終端機/CLI
     ▼
     S (黑色)
```

這創造了：
- **視覺清晰度**：立即傳達「程式碼導覽指南」
- **象徵意義**：指南針 = 找到方向；`>` = 程式碼/終端機
- **可擴展性**：在 16px 到 512px 的任何尺寸都能運作
- **品牌一致性**：以極簡色彩搭配開發者工具美學

## 色彩系統

### 色彩方案

| 色彩 | Hex | RGB | 用途 |
|-------|-----|-----|-------|
| 黑色（主色） | `#000000` | 0, 0, 0 | 主要筆畫、文字、南針 |
| 白色（背景） | `#FFFFFF` | 255, 255, 255 | 淺色背景 |
| 灰色（次要色） | `#6B7280` | 107, 114, 128 | 次要刻度標記、次要文字 |
| 亮綠色（強調色） | `#22C55E` | 34, 197, 94 | 北針、中心點、強調線 |
| 近黑色（深色背景） | `#0A0A0A` | 10, 10, 10 | 深色模式背景 |

### 對比度（WCAG）

- 黑色在白色上：**21:1** AAA
- 灰色在白色上：**4.6:1** AA
- 綠色在白色上：**3.2:1**（僅用於裝飾，不用於文字）
- 白色在深色上：**19.5:1** AAA

### 強調色規則

**亮綠色 (#22C55E) 僅保留用於強調：**
- 指南針北針
- 中心點
- 強調底線/邊框
- 絕不作為背景色
- 絕不用於正文文字

## 圖示詳細資訊

### 指南針規格

指南針標誌由以下幾何元素構成：

```
元素                 | 筆畫/填充      | 色彩
--------------------|----------------|------------------
外環                | 3px 筆畫       | 黑色 / 白色（深色模式）
北刻度              | 2.5px 筆畫     | 黑色 / 白色（深色模式）
其他主方位刻度      | 2px 筆畫       | 灰色 / 白色 50%（深色模式）
副方位刻度          | 1.5px 筆畫     | 灰色 / 白色 40%（深色模式）
北針                | 填充多邊形     | #22C55E（始終為綠色）
南針                | 填充多邊形     | 黑色 / 白色（深色模式）
> 括號              | 3px 筆畫       | 黑色 / 白色（深色模式）
中心點              | 填充圓形       | #22C55E（始終為綠色）
```

### 尺寸漸進

```
16px  → 僅環形 + 指針 + 箭頭（極簡）
32px  → 新增主方位刻度標記
64px  → 新增副方位刻度標記
128px → 完整細節，所有元素清晰
256px → 最大細節，粗筆畫
```

## 尺寸指南

### Logo 尺寸

- **最小**：200px 寬（用於網頁）
- **建議**：520px（原始尺寸）
- **最大**：無限制（向量格式）
- **長寬比**：約 4.3:1（寬:高）

### 圖示尺寸

- **最小**：16px（favicon）
- **建議**：64-256px（應用程式、頭像）
- **最大**：無限制（向量格式）
- **長寬比**：1:1（方形）

## 自訂

### 變更強調色

建立不同強調色的變體：

1. 將所有 `#22C55E` 替換為您的強調色
2. 確保裝飾元素的對比度維持在 3:1 以上
3. 保持黑色/白色/灰色結構不變

### 縮放

```css
svg {
  width: 256px;
  height: 256px;
}
```

SVG 透過 viewBox 自動縮放 — 無需變換。
